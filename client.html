Installation Guide

1

Download File

Buka tab My Add-ons dan klik tombol Download Latest. Tunggu hingga file .mcpack selesai diunduh ke perangkat Anda.

2

Install ke Minecraft

Klik ganda (double-click) file .mcpack yang baru saja diunduh. Minecraft akan otomatis terbuka dan memunculkan tulisan "Import Started" di bagian atas layar.

3

Pasang di World

Masuk ke menu Edit World. Pastikan Anda memasang Behavior Pack dan Resource Pack Admin Suite. Aktifkan juga opsi Beta APIs di bagian Experiments.

4

Verifikasi In-Game

Masuk ke dalam world Anda. Gunakan item Admin atau ketik command /as:info. Sistem akan meminta Anda memasukkan 16-Digit License Key dari tab Licenses. Selesai!
